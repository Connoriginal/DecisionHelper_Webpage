Contribute


All kinds of contributions (enhancements, new features, documentation & code improvements, issues & bugs reporting) are welcome.

Issue


When you create an issue to report a bug or ask a question, please include your:
  • Operating system with version And all others related information that are susceptible to help us.

Pull Requests


All pull requests must be done on the pull_request branch. Before a pull request:
  • There is no test yet so verify that your code is running well.

  • Don't forget to update user and/or developer documentation if it's necessary
